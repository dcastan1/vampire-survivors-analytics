# Vampire Survivors — Gaming Analytics

Proyecto personal de análisis de datos aplicado al videojuego Vampire Survivors.

## ¿Qué contiene este proyecto?
- Dataset real de 25 partidas registradas manualmente
- Análisis de rendimiento por personaje y mapa
- Dashboard de visualización con matplotlib
- Reporte exportado en HTML

## Tecnologías usadas
- Python · Pandas · Matplotlib · Jupyter Notebook

## Hallazgos principales
- Mejor personaje: Mortaccio (71% win rate, mayor oro promedio)
- Mejor mapa: Mad Forest (83% win rate)
- Combinación óptima: Mortaccio + Mad Forest
